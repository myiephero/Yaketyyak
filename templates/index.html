Features

Everything you need to
feel at home in the terminal.

No prior experience required. Just open it and start typing.

Real shell. Real results.

Not a sandbox. Your commands run on your actual system with a real PTY. Files you create, errors you see, and outputs you get are all real.

Git Translator.

Type /git <url> to analyze any GitHub repo. See stars, forks, license, and a quality score instantly.

500+ commands. Instant.

The built-in knowledge base explains 500+ commands and 50+ error patterns in milliseconds — no internet, no AI, no latency.

AI when you need it.

Local AI via Ollama handles anything the knowledge base can't. Runs on your machine. No cloud, no cost, no data leaving your computer.

25 starter commands.

Type try 1 through try 25 to run safe, guided commands that teach you the terminal step by step.

Private by default. Two built-in themes.

No accounts, no tracking, no telemetry. Everything runs locally. Switch between a hacker-green Terminal look and a sleek glassmorphic Glass theme with Ctrl+S. Your preference is remembered.

How It Works

Running in 60 seconds.

No configuration. No installer. No terminal experience needed.

02

Open

Double-click. On Mac it opens Terminal automatically. On Linux it shows in your app menu.

03

Learn

Type try 1 to run your first command. The right panel explains everything in plain language.
