Shell
$ ls -la

drwxr-xr-x 5 user staff 160 Mar 6
-rw-r--r-- 1 user staff 2048 app.py
drwxr-xr-x 3 user staff 96 node_modules
$ git status
On branch main
Changes not staged for commit:
  modified: app.py
$ pip install flask
ERROR: No matching distribution
Translation
[Local KB] command
This lists all files and folders with details. The letters like "drwxr-xr-x" show permissions — who can read, write, or run each file. The "5 user staff" means you own these files.
[Local KB] command
You have changes to app.py that haven't been saved to git yet. Run "git add app.py" then "git commit" to save them.
[Ollama AI] error
The package "flask" can't be found. This usually means your Python version is too old or pip needs updating. Try: pip3 install flask
